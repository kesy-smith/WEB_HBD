## 🧠 Comprendre le Code

### Pour Modifier les Souhaits
1. Ouvrez [js/wishes-generator.js](js/wishes-generator.js)
2. Trouvez `this.wishes.emotion`
3. Ajoutez votre souhait au tableau

### Pour Modifier les Couleurs
1. Ouvrez [css/styles.css](css/styles.css)
2. Modifiez `:root { --primary-neon: ... }`
3. Modifiez [js/three-particles.js](js/three-particles.js) emotionalColors

### Pour Modifier les Paramètres
1. Ouvrez [js/config.js](js/config.js)
2. Modifiez le CONFIG object
3. Redémarrez l'app

### Pour Déboguer
1. Appuyez sur F12
2. Allez à "Console"
3. Copiez les commandes de [ADVANCED_API.md](ADVANCED_API.md)
